Most used DevOps Tools

Here are the most used DevOps Tools:

  • Cloud Providers (e.g., AWS, Azure, GCP)
  • Version Control (e.g., Git, GitHub)
  • CI/CD Pipelines (e.g., Jenkins, GitLab CI/CD)
  • Configuration Management (e.g., Ansible, Chef, Puppet)
  • Containerization & Orchestration (e.g., Docker, Kubernetes, Helm)
  • Infrastructure as Code (IaC) (e.g., Terraform, AWS)
  • Monitoring & Logging (e.g., Prometheus, Grafana, ELK Stack)
  • Collaboration & Communication (e.g., Slack, JIRA)
  • Security (e.g., SonarQube, Snyk)

The previous tools mentioned are not the only ones, there are a lot more tools, this is just a generic list with some of the most used tools. You should search more tools if curious.
